 # Pembuatan Workflow KNIME Menggunakan Data Reading Interest
 
 ---

Data dibuat dengan format "Visual Analysis of Sales Data"

Berikut adalah langkah-langkah untuk memvisualisasi Data:

---

### 1. Mengimport Data
Memasukkan data "Reading Interest" dalam workspace Knime dengan CVS Reader.

---

### 2. Membersihkan Data
* Memilih data tertentu.
* Menghilangkan data yang kosong.
* Menghilangkan data yang kembar.

---

### 3. Visualisasi Data
Data divisualisasi dengan:
* Bar Chart
* Histogram
* Scatter Plot
